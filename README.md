# Bees Live 🐝

Ein einfaches 2D-Top-Down-Browserspiel, in dem du eine Biene von Blume zu Blume steuerst, Blütenstaub sammelst und Punkte erzielst – bevor der Timer abläuft!

## Spielprinzip

- Steuere deine Biene über eine Blumenwiese (WASD / Pfeiltasten)
- Sammle Pollen von Blumen ein und erhalte Punkte
- Ein Countdown-Timer gibt die Spielzeit vor
- Am Ende wird dein Punktestand angezeigt
